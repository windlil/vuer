## Start

We recommend using degit to clone template.
```bash
npx degit windlil/vuer project_name

cd project_name

pnpm i
```

## Features

- Vue3
- Vite5
- Tailwindcss
- Vue-Router
- Pinia
- iconify
- dark mode

## Scripts

```
// init project and delete redundant files
pnpm clean
```
